# Python Foundations

This repository is for learning and practicing Python fundamentals.  

## Focus Areas
- Logic patterns (loops, conditionals, decision flow)
- Data structures (lists, dictionaries, sets, tuples, strings)
- Functions and classes (basic OOP)
- Algorithms (Fibonacci, prime numbers, sorting, recursion)

## How to Use
1. Clone the repo.
2. Navigate to the folder of interest.
3. Open Python files and run exercises.
4. Update your glossary with any new concepts you learn.

## Mini-Apps
Saturday exercises combine multiple concepts into small applications.
